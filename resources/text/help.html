photoTweaker

PhotoTweaker is a small photo editor that can be integrated in a simple photo and picture management workflow.

It's aim is to facilitate a fast, simple and ergonomic manipulation of invividual pictures.

Basic concepts

  • As few click as possible.
  • No dialogs shown while you're working on the image.
  • If a tool needs the user to define a set of values, those will be defined in Edit > Prefernces and multiple buttons will be shown, each with a specific value.
  • Only the features that are strictly necessary get implemented.
  • Effects are implemented as plugins and you're welcome to propose and add new ones!

Opening an image

photoTweaker can open PNG and JPG images. The normal way to open an image is to set the image as a parameter when launching photoTweaker.

Currently, you can also use File > Open, but it's likey that the menu entry will be removed in future version and replaced by the File > Revert command.

You can only open one image at a time.

Saving the image

File > Save lets you overwrite the image and save the changes you've done

If you want to save the image under a new name, you have duplicate the image before opening it with photoTweaker

Zooming

photoTweaker does not really have a zoom. The image gets automatically shrinked to fit the window size: you can resize the window to see a bigger or smaller view of your image.

photoTweaker won't zoom further in than a 1:1 ratio.

Cropping

Cropping is as simple as selecting a region: on save only the selected area will be kept.

If nothing is selected, the full image will be saved.

Scaling

If you click on one of the Scaling buttons, on save the image will be shrinked to the given measurement.

You can add new measurements in Edit > Preferences

Currently, the size always refers to the "biggest" measurment: the width for horizontal images and the height for vertial ones. In the future finer settings will be possible (always horizontal...)

Undo

You can always undo / redo the last step.

Scripts

You can use photoTweaker inside of scripts. In the scripts/ directory you can find some examples.
